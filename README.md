# Rohit Gupta Portfolio

A personal portfolio website for showcasing my projects, skills, experience, resume, and contact information. The site is built as a lightweight static website using HTML, CSS, and JavaScript, and is deployed with GitHub Pages.

## Live Demo

Visit the deployed portfolio here:

https://rohitguptaxo.github.io/my-portfolio/

## Repository

GitHub repository:

https://github.com/rohitguptaxo/my-portfolio

## Features

- Responsive portfolio layout
- About section with profile image and animated role text
- Experience and education highlights
- Project showcase with live demo and GitHub links
- Skills section
- Resume download link
- Contact section with email and social links

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
- GitHub Pages for deployment

## How to Clone and Run Locally

1. Clone the repository:

```bash
git clone https://github.com/rohitguptaxo/my-portfolio.git
```

2. Open the project folder:

```bash
cd my-portfolio
```

3. Open `index.html` in your browser.

Because this is a static website, no package installation or build step is required.

## Deployment

This portfolio is deployed using GitHub Pages from the `main` branch.
